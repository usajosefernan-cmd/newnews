# Antigravity — Reglas de Comportamiento del Proyecto

## 🚨 CÓDIGO ROJO — PROHIBICIÓN ABSOLUTA DE SIMULACIONES
- **NADA MOCKEADO, SIMULADO O INVENTADO**: Está terminantemente prohibido mockear, simular, inventar o inyectar datos ficticios (como números de visualizaciones, likes, métricas de engagement, enlaces falsos, veredictos predeterminados o respuestas simuladas de la IA).
- Todo el motor de Matiza y el pipeline de análisis de Hermes deben operar con datos 100% reales obtenidos de forma directa y orgánica de las plataformas (YouTube, TikTok, Instagram, X, Telegram y prensa oficial).
- Si no hay datos oficiales de engagement disponibles debido a bloqueos, se reportará explícitamente como "No disponible" o se omitirá el campo de forma limpia, pero **bajo ninguna circunstancia** se inyectarán valores numéricos falsificados o simulados.
